Время перед отсылкой предупреждения
Если доставка сообщения не прошла по некоторым временным причинам (например сервер - адресат был выключен), fail2ban fail2ban сохранит сообщение в очереди, периодически пытаясь его отослать. По прошествии указанного в данном параметре времени, сообщение с предупреждением о задержке в доставке почты будет отослано обратно.
