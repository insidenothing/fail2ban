Espace Disque Minimum
Si moins que la quantité d'espace disque spécifié est disponible, Fail2ban refusera de mettre dans sa file d'attente les messages entrant des autres systèmes. Utile pour prévenir Fail2ban d'utiliser tout l'espace disque sur le système de fichier dans lequel les couriels sont placé dans la file d'attente.
