Максимальная средняя загрузка для приема
Определяет среднюю загрузку выше которой fail2ban больше не принимает сообщения по SMTP, заставляя другие почтовые серверы ставить сообщения в очередь для более поздней доставки. Данный параметр полезен для предотвращения перегрузки вашей системы высоким почтовым трафиком.
